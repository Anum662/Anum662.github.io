Chrono App — Route Planning Redesign

A step‑by‑step usability evaluation of Montreal's transit trip planner.

Task Flow Overview

This walkthrough documents each stage of planning a trip using the Chrono App. Each screen illustrates the user action and the system’s feedback, highlighting usability challenges and opportunities for improvement.

Steps 1–5: Beginning the trip‑planning task — selecting destination, starting a search query, and viewing initial route options.

Steps 6–13: Entering departure address, adjusting date and time, and navigating the calendar interface.

Steps 14–18: Modifying arrival time, confirming input details, and viewing the finalized list of route options.

Heuristic Evaluation Table

This table summarizes insights using Nielsen’s heuristics: visibility of system status, match between system and real world, consistency, user control, and feedback.

Heuristic Evaluation: Steps 1–6, focusing on labeling clarity, conceptual model alignment, and visibility of available options.

Heuristic Evaluation: Steps 7–13, identifying inconsistencies in wording, mixed languages, and confusing button states.

Heuristic Evaluation: Steps 14–18, covering time input issues, confirmation interactions, and improvements to route selection screens.
